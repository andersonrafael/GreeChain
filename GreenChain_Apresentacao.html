O Problema
MERCADO
OPACO
E FRÁGIL

O mercado global de créditos de carbono movimenta bilhões, mas opera sem transparência ou auditabilidade real.

Nossa Solução
Cada compensação de CO₂ vira um NFT único e imutável na blockchain — verificável por qualquer pessoa, para sempre.
🔴
Opacidade Total
Compradores não conseguem verificar se um crédito é legítimo ou se representa compensação real.
⚠️
Double-Counting
O mesmo crédito pode ser vendido múltiplas vezes para compradores diferentes sem que ninguém saiba.
🏦
Centralização
Terceiros privados controlam emissão, verificação e invalidação de créditos sem accountability.
GreenChain: blockchain pública + NFT único por certificado + DAO comunitária = transparência total e impossibilidade de fraude.

GreenStaking.sol — stake() + oracle
// APR dinâmico via Chainlink
function _calculateRewards(
  uint256 amount, uint256 since
) internal view returns (uint256) {
  uint256 apr = BASE_APR_BPS; // 10%
  int256 price = getEthUsdPrice();
  if (price > int256(ETH_BONUS_THRESHOLD))
    apr += BONUS_APR_BPS; // +5% se ETH > $2k
  return (amount * apr * elapsed)
         / (SECONDS_PER_YEAR * 10_000);
}

// Checks-Effects-Interactions
function unstake(uint256 amount)
  external nonReentrant {
  _accumulateRewards(msg.sender); // 1
  stakes[msg.sender].amount -= amount; // 2
  stakingToken.safeTransfer(msg.sender, amount); // 3
}
GreenDAO.sol — vote()
function vote(uint256 proposalId, bool support)
  external {
  require(!hasVoted[proposalId][msg.sender],
    "already voted"); // anti double-vote
  uint256 weight = govToken.balanceOf(msg.sender);
  hasVoted[proposalId][msg.sender] = true;
  if (support) p.votesFor += weight;
  else p.votesAgainst += weight; // peso = saldo GRN
  emit VoteCast(proposalId, msg.sender, support, weight);
}
